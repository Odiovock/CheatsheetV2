Semantic HTML

Semantic HTML is used to give meaning to a page through elements that provide context to their content.

Basic structure tags
Tag Use
<header> Usually contain navigational links and introductory content
<nav> Define a block of navigation links
<main> Contains the dominant content within the webpage
<footer> Contains information such as the contacct information, terms of use, etc.
Body semantic tags
Tag Use
<section> Separate elements in a documents that have the same theme
<article> Hold the content that makes sens on it's own
<aside> Contains information relating to the main content, but that is not required to understand the dominant information
<figure> Encapsulates all types of media
<figcaption> Used for the figure tag's description

Media in HTML

There are a lot of ways to implement media in a web page. You can add videos, images, gifs and more!

Media tags
Tag Media Use
<img> Images Link images from the website structure with the src attribute or from another website using src with url("")
<video> Video files Link video files using the src attribute
<audio> Audio files Link audio files using the <source> tag and the src attribute
<embed> Any media type *This type of element is deprecated and is shown for legacy purposes.* Link any type of media using the src attribute
